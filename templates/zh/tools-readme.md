# 工具索引

只在这个目录中放置高信号、可复用的辅助工具。

## 什么适合放在这里

- 能节省 agents 重复劳动的脚本
- 稳定的查询辅助工具
- 与重复 workflow 绑定的操作手册
- 支持 EDITH maintenance 或 retrieval 的小工具

## 什么不适合放在这里

- 随机的一次性实验
- 没有 EDITH-specific 价值的通用工具
- 更适合保留在 source repos 中的大型操作手册
